Press UP or Down on your keyboard to move slide into view. Double click anywhere to toggle fullscreen on or off.

What is ViewJS you say?

ViewJS - is a HTML5 & JavaScript is a web presentation plugin, that is a light weight. Lightweight and simplicity is the main point of ViewJS. Just include theview.js file at the bottom of your HTML file. The JS code looks for all class="slide" instances and creates slides for each.

You say its lightweight so what do I get?

  • css/content.css - content style (change it to your needs of your design)
  • css/layout.css - include layout.css into your HTML
  • js/view.min.js - all the magic happens here

How do I use this bad boy?

  1. Create an HTML file and include the following in the head section: css/layout.css. Next add css/content.css for your own styling.
  2. Create a script tag right above the closing body tag, like so: script src="js/view.min.js".
  3. Mark up the HTML file with your content you want to be displayed in the presentation. Slides are created using the class slide, like so: div class="slide".
