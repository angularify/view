What is it view.js?

View.js - is html/javascript very very lightweight framework for building presentation. Lightweight and simplicity is main point of view.js. Just include view.js file to your html file, fill content, write css design for your presentation as you choose and that's all, it's ready to use.

What is in?

  • css/content.css - content style (change it to your needs of your design)
  • css/layout.css - presentation layout (do not edit this!)
  • js/view.js - main view.js functional (do not edit this!)

How to use?

First of all create html file which will be future presentation, like this index.html. Incude to your html file
css/layout.css and css/content.css in head section and js/view.js in the end of body. Now you can
create the slides. Every slide is div element with class = "slide" and also with some presentation content in it.
Just put some divs with class="slide" into your html file and edit content.css for your design needs.
